# readtest-ai


这是一个浏览器插件，目前国内网站存在大量广告以及对网站阅读设置不利于浏览网页，降低网页的可读性。
随着ai时代的到来，我们无法实时对网页与ai交互，插件将集成ai和笔记功能方便浏览网页中的记录和分析。</br>
<img width="700" height="300" alt="image" src="https://github.com/user-attachments/assets/b7c95017-5269-4044-9ced-4bd1bb8094f7" />



使用教程：
1、打开Chrome浏览器或edge浏览器</br>
2、打开插件内容部分，打开开发者模式</br>
3、点击加载未打包的扩展程序</br>
<img width="700" height="300" alt="image" src="https://github.com/user-attachments/assets/62d40c9c-e88d-4b94-8e3b-167b48411187" />
4、将解压好的安装包导入即可</br>

ai功能的使用：</br>
编辑content.js里面放着关于ai配置的功能代码，更新自己的即可</br>
